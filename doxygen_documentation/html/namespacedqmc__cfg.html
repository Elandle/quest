Stores the names of input parameters.

Must be of length N_Param. See PARAM_DVAL for the default value of a parameter, which is used if not specified on input.

Parameter names: 

 HSF     (integer)      : specifies the method in which the Hubbard Stratonovich field (HSF) is input or generated.
 HSFin   (file name)    : file name to read HSF from if HSF is set to HSF_FROM_FILE.
 HSFout  (file name)    : file name to output HSF to.
 HSFtype (integer)      : 0 if HSF is discrete, 1 if HSF is continuous.
 L       (integer)      : number of imaginary time slices.
 U       (real)         : potential energy.
 accept  (integer)      : accept counts.
 bcond   (real, dim(3)) : boundary conditions.
 debug                  : flag for debug information output.
