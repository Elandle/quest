147  ! For a Nx*Ny 2D rectangular lattice.
148  !
149  ! 1. The sites are numbered from 1, which is the site on the
150  ! south-west corner. The numbering is row major, which means
151  ! it increases along x direction first (left to right) and
152  ! then y-direction (bottom up).
153  !
154  ! 2. Adjacency (T) has exact 4 elements per site: left
155  ! right, up and down. Since the adjacency is cyclic,
156  ! the code has had spacial treat for boundary sites.
157  ! *** The way it computed here is not satisfied the
158  ! 'checkboard' order. should change it later.
159  !
160  ! 3. The number of unique distance is computed as follows.
161  ! Let long = max(Nx, Ny) and short = min(Nx, Ny).
162  ! The distinct distance sites form a trapezoid
163  ! The bottom is (long/2+1), the top is ((long-short)/2+1)
164  ! and the height is (short/2+1). Therefore,
165  !
166  ! nClass = (short/2+1)*(long-short/2+2)/2
167  !
168  ! 4. The phase (P) is computed in the rules that
169  ! (a) adjacent sites have opposite phase.
170  ! (b) site 1 is phased +.
